# AI-Based Student Career Path Recommendation System

A machine learning web application that recommends the best career path for a student based on their skills, interests, scores, and preferences.

---

## Project Overview

Students often struggle to choose a career path that matches their strengths and interests. This system uses a **Random Forest Classifier** trained on a synthetic dataset to predict the most suitable career path out of 10 options.

The user fills in a web form with their academic scores, interest levels, and preferences. The ML model processes this data and returns a career recommendation along with confidence score, skills to improve, a learning roadmap, and project ideas.

## Dataset

The dataset `student_career_dataset.csv` is generated by `generate_dataset.py`. It contains 200 rows (20 per career path) with the following columns:

**Numeric features (scores):** `math_score`, `programming_score`, `communication_score`, `problem_solving_score`

**Interest levels (1-10):** `web_interest`, `mobile_interest`, `ai_interest`, `database_interest`, `networking_interest`, `cloud_interest`, `design_interest`

**Categorical features:** `preferred_subject`, `career_goal`

**Target variable:** `recommended_path` (one of 10 career paths)

The dataset is balanced across these career paths:
- Frontend Developer
- Backend Developer
- Full Stack Developer
- Mobile App Developer
- Data Analyst
- Machine Learning Engineer
- Cybersecurity Analyst
- Cloud/DevOps Engineer
- UI/UX Designer
- Database Administrator

---

## ML Algorithm

**RandomForestClassifier** is used as the primary classification model.

- **n_estimators:** 200 decision trees
- **Train/test split:** 80/20 with stratification
- **Categorical encoding:** LabelEncoder for subject, career goal, and target
- **Evaluation:** Accuracy score and classification report

Random Forest is chosen because it:
- Handles mixed feature types well
- Reduces overfitting through ensemble averaging
- Works well with smaller datasets
- Provides feature importance insights

## Installation & Setup

### Prerequisites
- Python 3.8 or higher
- pip (Python package manager)

### Step 1: Clone or download the project

```bash
cd student-career-recommender
```

### Step 2: Create a virtual environment

**macOS/Linux:**
```bash
python -m venv venv
source venv/bin/activate
```

**Windows:**
```bash
python -m venv venv
venv\Scripts\activate
```

### Step 3: Install dependencies

```bash
pip install -r requirements.txt
```

### Step 4: Generate the dataset

```bash
python generate_dataset.py
```

This creates `dataset/student_career_dataset.csv` with 200 rows of synthetic data.

### Step 5: Train the model

```bash
python train_model.py
```

This trains the Random Forest model and saves:
- `models/career_model.pkl` - the trained model
- `models/label_encoder.pkl` - the target label encoder

### Step 6: Run the Flask application

```bash
python app.py
```

### Step 7: Open in browser

Navigate to: **http://127.0.0.1:5000**

## Common Errors & Fixes

| Error | Fix |
|-------|-----|
| `ModuleNotFoundError: No module named 'flask'` | Run `pip install -r requirements.txt` |
| `FileNotFoundError: Dataset not found` | Run `python generate_dataset.py` first |
| `FileNotFoundError: Model files not found` | Run `python train_model.py` first |
| `Address already in use` | Change port in `app.py` or kill the existing process |
| `Model accuracy seems low` | Regenerate dataset with `python generate_dataset.py` and retrain |
